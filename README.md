# Save the Novel

**The Reader Speaks.**

A web project by S.E. Elkins to gather and broadcast the experience of reading — not for information, but for connection, belonging, empathy, nuance.

**Launch: September 1, 2026**

---

## Visual Explainer

Open `docs/index.html` in a browser for an interactive visual overview of the entire project architecture, content model, timeline, and tech stack.

**GitHub Pages:** This repo is structured so `docs/` can be served directly via GitHub Pages.
